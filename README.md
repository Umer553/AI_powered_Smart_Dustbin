# 🗑️ AI Powered Smart Dustbin

An **AI-based Smart Waste Management System** that automatically detects, classifies, and manages waste using computer vision and embedded systems. The system helps improve waste segregation and promotes efficient recycling by identifying waste types and directing them into appropriate bins.

This project combines **Artificial Intelligence, Computer Vision, and IoT** to build an intelligent dustbin capable of recognizing waste materials and enabling smart waste management.

---

# 📌 Project Overview

Traditional waste disposal methods rely heavily on manual segregation, which is inefficient and often leads to environmental pollution.

The **AI Powered Smart Dustbin** addresses this issue by using a **deep learning model and sensors** to automatically detect and classify waste items into predefined categories such as:

- Recyclable Waste  
- Organic Waste  
- Plastic Waste  
- Metal Waste  

Once the waste is detected, the system automatically **opens the appropriate compartment**, improving recycling efficiency and reducing human intervention.

# 🧠 System Workflow

The system works in the following steps:

1. A user approaches the dustbin.
2. The sensor detects the presence of an object.
3. The camera captures an image of the waste item.
4. The AI model processes the captured image.
5. The system classifies the waste into a specific category.
6. The corresponding compartment opens automatically.
7. Waste is disposed into the correct section.

---

# 🧪 Model Training

The waste classification model was trained using a **Convolutional Neural Network (CNN)** on a dataset containing images of different waste materials.

Steps involved:

1. Dataset collection and cleaning  
2. Image preprocessing and augmentation  
3. Model architecture design  
4. Model training and validation  
5. Performance evaluation  

Evaluation metrics include:

- Accuracy  
- Precision  
- Recall  
- Confusion Matrix
